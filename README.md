# Age in Days Calculator

This is a simple Python project that calculates your age in days based on your age in years.

Although this project is simple, I created it because I enjoy this domain and I like learning step by step and sharing my progress.  
I believe that small projects are important to build strong foundations.

## Features
- Takes age in years as input
- Calculates age in days
- Simple and beginner-friendly

## How to Run
1. Make sure Python is installed on your computer
2. Download or clone this repository
3. Run the script using:
   python age_calculator.py
4. Enter your age in years when asked

## Example
Input: 20  
Output: You have lived for approximately 7300 days
